# Docker Operations

**name:** docker-operations

**Overview:** Docker CLI patterns for containers, images, networks, volumes (local and remote hosts)

**Common patterns:**
- `docker ps`
- `docker logs`
- `docker exec` 
- `docker-compose up/down`
- Image builds
- Registry operations

**Safety notes:**
- Confirm before destructive ops (rm, prune)
- Be explicit about production vs dev
