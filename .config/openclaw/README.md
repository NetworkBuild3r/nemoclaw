# OpenClaw host env (Vault bootstrap)

This directory holds **machine-local** files used by systemd and scripts:

| File | Purpose |
|------|---------|
| `vault.env` | `VAULT_ADDR` + `VAULT_TOKEN` for `ExecStartPre` and CLI (`mode 600`) |
| `aws-bedrock.env` | Generated by `scripts/vault-sync-aws-bedrock-env.sh` from Vault (`mode 600`) |

**Canonical path:** `/home/bnelson/nemoclaw/.config/openclaw/`

`~/.config/openclaw` is a **symlink** here so older docs and tools keep working. Do not commit secrets; this tree is gitignored except for this README.
