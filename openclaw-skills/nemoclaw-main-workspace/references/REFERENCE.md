# main / default workspace

`workspace-default/README.md` explains this folder is used when `agents.defaults.workspace` points here.

For GitOps specialists, prefer opening `agents/gitops-chief`, `agents/github-bob`, etc. OpenClaw loads all skills from `openclaw-skills/` via `skills.load.extraDirs` regardless; subfolder workspaces still set the agent workspace path for tools and files.
